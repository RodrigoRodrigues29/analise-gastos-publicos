# Análise de Gastos Públicos - Portal da Transparência

Projeto de análise de dados utilizando Python com dados do Portal da Transparência do Governo Federal.

## Objetivo
Analisar como os gastos públicos estão distribuídos entre os órgãos ao longo do ano.

## Dados utilizados
- Órgão superior
- Valores: empenhado, liquidado e pago
- Período (mês/ano)

## Ferramentas
- Python
- Pandas
- Matplotlib
- Google Colab

## Análises realizadas
- Ranking dos órgãos que mais gastam (valor pago)
- Evolução dos gastos ao longo dos meses
- Comparação: Ministério da Fazenda vs demais órgãos

## Principais insights
- O Ministério da Fazenda concentra grande parte dos gastos
- Existe variação significativa ao longo dos meses
- Outros órgãos apresentam valores bem menores em comparação

## Como executar
1. Baixar o arquivo `.ipynb`
2. Abrir no Google Colab ou Jupyter Notebook
3. Executar as células
